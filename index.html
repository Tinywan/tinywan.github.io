Laravel

全栈框架，功能齐全。认证、数据库等功能一应俱全。

composer create-project laravel/laravel app

Symfony

适用于复杂应用程序的高性能框架。模块化和灵活。

symfony new my_project --webapp

Slim

适用于API和微服务的微框架。快速和极简。

composer create-project slim/slim-skeleton app

WordPress

驱动43%网站的CMS。可扩展和用户友好。

wp core download && wp core install

API Platform

分钟内构建REST和GraphQL API。基于Symfony构建。

composer create-project api-platform/api-platform

Webman

composer create-project workerman/webman:~2.0
